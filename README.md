# Dragon Repeller Game

**Dragon Repeller** is a beginner-friendly web development project that combines **HTML**, **CSS**, and **JavaScript** to create an interactive, terminal-style game that can run locally in a web browser.

---

## Overview

- **HTML**: Contains all the necessary elements and buttons to make the game interactive. Serves as the structural base.  
- **CSS**: Styles the game to resemble a terminal-like interface for a polished look.  
- **JavaScript**: Controls game logic, manipulates button texts/functions, and makes the game dynamic.

---

## Features

- Multiple **monsters** with different levels and health to challenge the player.  
- Interactive **buttons** to navigate locations, fight monsters, buy health, and upgrade weapons.  
- Uses **objects, arrays, variables, constants, and functions** for game interactivity.  
- **Hidden Easter egg** feature to make gameplay more fun.  
- Open-source and free to play!

---

## How to Play

1. Download the repository:

2. Open index.html in your web browser.

3. Use the on-screen buttons to explore the different locations.

4 Discover the hidden Easter egg!
